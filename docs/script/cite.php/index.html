Cite

Learn how to cite ImageMagick in your work. Choose from these citation formats:

Citation in APA style

ImageMagick Studio LLC. (2023). ImageMagick. Retrieved from https://imagemagick.org

Citation in Vancouver style

1. ImageMagick Studio LLC. ImageMagick [Internet]. 2023. Available from: https://imagemagick.org

Citation in Harvard style

ImageMagick Studio LLC, 2023. ImageMagick, Available at: https://imagemagick.org.

Citation in Bibtex format

{% raw %}@software{imagemagick,
  author = {{ImageMagick Studio LLC}},
  title = {ImageMagick},
  url = {https://imagemagick.org},
  version = {7.0.10},
  date = {2023-01-04},
}{% endraw %}
